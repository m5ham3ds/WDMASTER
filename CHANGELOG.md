## [1.0.0] - 2026-04-25

### 🎉 الإصدار الأول

#### 🚀 ميزات مضافة
- 🎴 **توليد البطاقات**: إمكانية توليد حتى 1000 بطاقة دفعة واحدة.
- 🧪 **اختبار البطاقات**: محاكاة اختبار تلقائي عبر WebView مع دعم MikroTik والنماذج العادية.
- 📡 **إدارة الراوترات**: إضافة وتعديل وحذف راوترات متعددة مع إعدادات CSS Selectors مخصصة.
- 🔔 **خدمة خلفية**: تشغيل الاختبار في الخلفية مع إشعارات تفاعلية (إيقاف/استئناف/إلغاء).
- 📊 **إحصائيات حية**: عرض مباشر لعدد المحاولات، النجاح، الفشل، السرعة، والوقت المتبقي.
- 📜 **سجل طرفي**: عرض مباشر لجميع الأحداث بتنسيق Terminal مميز.
- 📤 **تصدير النتائج**: حفظ النتائج بصيغة JSON.
- 🌐 **دعم اللغات**: العربية 🇸🇦 والإنجليزية 🇬🇧 كاملة.
- 🎨 **ثيمات متعددة**: فاتح ☀️، داكن 🌙، وافتراضي النظام 📱.
- 🧵 **تعدد المهام**: دعم 1-5 مسارات متوازية.

#### 🗄️ قاعدة البيانات
- 🏛️ **Room Database**: 5 جداول (`cards`, `router_profiles`, `test_results`, `successful_patterns`, `test_sessions`).
- 📦 **DataStore**: لحفظ التفضيلات العامة وإعدادات الثيم واللغة.

#### 🏗️ المعمارية
- 🧱 **MVVM + Repository Pattern**: طبقات نظيفة: Presentation → Domain → Data.
- 💉 **Koin**: حقن تبعيات خفيف وآمن.
- 🧭 **Navigation Component**: تنقل بين 5 وجهات رئيسية.

#### 🤖 CI/CD
- 🔄 **GitHub Actions**: بناء تلقائي، اختبارات، وإصدارات.
