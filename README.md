# 📶 WiFi Card Master Pro

> تطبيق Android احترافي لتوليد واختبار بطاقات WiFi على أجهزة MikroTik والراوترات المحلية.  
> يتميز بواجهة عصرية، ومحرك اختبار متطور عبر WebView، وإشعارات تفاعلية، ودعم كامل للغة العربية.

## 🚀 البداية السريعة

### المتطلبات الأساسية

| الأداة | الإصدار المطلوب |
|:---:|:---:|
| ☕ JDK | 17 |
| 🛠️ Android Studio | Hedgehog (2023.1) أو أحدث |
| 📦 Gradle | 8.2 |
| 📱 Android SDK | API 34 |

### التثبيت والتشغيل

```bash
# 1. استنساخ المستودع
git clone https://github.com/username/WiFiCardMasterPro.git

# 2. فتح المشروع في Android Studio
# File → Open → اختر مجلد WiFiCardMasterPro

# 3. مزامنة Gradle
./gradlew sync

# 4. بناء التطبيق
./gradlew assembleDebug

# 5. تثبيت التطبيق على جهاز متصل
adb install app/build/outputs/apk/debug/app-debug.apk
```
